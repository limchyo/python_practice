12.2. 배스킨라빈스 게임 필승 인공지능 컴퓨터 만들기(단, 컴퓨터와 둘이서 게임시 적용)
> 2를 먼저 말하고 이후 30까지 4의 배수(6, 10, ... , 30)로 외치면 승리한다. 이 수들의 공톰점은 4로 나누었을때 나머지가 2이다. 즉, 상대가 1을 외치면 나는 3, 2는 2, 3은 1 식으로 합해서 4가 나오도록 함수를 설계하면 된다. 
```
def call_computer(call_status):
    if call_status % 4 == 0:
        computer_call = 2
    elif call_status % 4 == 1:
        computer_call = 1
    elif call_status % 4 == 3:
        computer_call = 3
    else:
        computer_call = random.randint(1, 3)

    return computer_call
```
> 굳이 내가 2를 빼앗아서 30까지 유지하여 이기는 것 아닌 이상 무조건 컴퓨터가 승리한다.
